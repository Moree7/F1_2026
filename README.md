Aplicación web para gestionar la temporada de Fórmula 1 2026. Pilotos, escuderías, circuitos, carreras, resultados y neumáticos.

---

## Qué puedes hacer

- Ver el Mundial de Pilotos y Constructores actualizado en tiempo real
- Consultar el calendario completo de los 24 GPs
- Registrar y editar resultados carrera a carrera
- Consultar el clima de cada circuito en tiempo real
- Sistema de login con roles

---

## Instalación

1. Copiar la carpeta `F1_2026` en `C:\xampp\htdocs\`
2. Crear la base de datos `f1_2026` en phpMyAdmin
3. Importar el archivo `f1_2026.sql`
4. Abrir `http://localhost/F1_2026/`
